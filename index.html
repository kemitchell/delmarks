    • Delmarks

    A node based command line tool for managing your delicious bookmarks.

    Project maintained by obihann Hosted on GitHub Pages — Theme by mattgraham

    Installation

    You can install us through npm

    npm install delmarks

    Usage

    Connect your account:

    delmarks connect USERNAME PASSWORD

    List bookmarks:

    delmarks ls

    Add bookmark:

    delmarks add http://github.com/

    Help:

    delmarks -h
